# NOOVEXOS

NOOVEXOS is a graphical desktop operating system built from scratch using C and x86 Assembly.

## Features

* Graphical Desktop Environment
* Window Manager
* Multitasking
* File Explorer
* Terminal
* Settings
* Task Manager
* Start Menu
* Update System
* Background Services
* Recovery Mode
* BSOD + Crash Logs
* File Permissions
* Memory Protection
* User / Kernel Separation
* Drivers
* Filesystem
* TCP/IP Networking
* DNS, UDP, HTTP Support

## Screenshots

Screenshots and boot demos can be found in this repository.

## Status

NOOVEXOS is currently under active development.

## Goal

Create a modern, lightweight desktop operating system.
